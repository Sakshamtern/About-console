PROJECT ON CONSOLE

Visual Studio is an integrated development environment (IDE). It is used to make websites. First you have to install visual Studio. Then you have to open it and make a folder for a project. Then open that project in the visual studio. To open html file you have to type index.html. For css style.css. And for javascript code.js.

A console screen is a combination of a display monitor and an input device. It is used to maintain or monitor the status of the network or computer. To open the console screen you have to open a blank page you made and right click on it. Click inspect and then go to console. To print something on console we have to type console.log and the var name in brackets and end with semicolon.
